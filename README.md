# 👶 新生儿起名系统

一款融合中华传统文化与现代技术的智能起名工具，为宝宝精心挑选寓意美好、五行平衡的好名字。

## ✨ 产品特色

- 📚 **古籍经典**：源自《诗经》《楚辞》《论语》等 12 部经典古籍
- 🔮 **命理分析**：完整的八字排盘、五行分析、八卦推演
- ⭐ **星座生肖**：融合西方星座学与东方生肖文化
- 🎨 **精美报告**：一键导出 PDF/Word 完整分析报告
- 💫 **书法展示**：马善政毛笔字体呈现名字美感

### 二、命理分析系统

#### 2.1 八字排盘分析

```
┌─────┬─────┬─────┬─────┐
│ 年柱 │ 月柱 │ 日柱 │ 时柱 │
├─────┼─────┼─────┼─────┤
│ 天干 │ 天干 │ 天干 │ 天干 │
│ 地支 │ 地支 │ 地支 │ 地支 │
│ 五行 │ 五行 │ 五行 │ 五行 │
└─────┴─────┴─────┴─────┘
```

**分析内容：**
- 四柱干支显示
- 天干地支五行属性
- 日主天干（代表命主自己）
- 八字强弱初步判断

#### 2.2 五行强弱分析

**可视化统计：**
- 🔲 **金**：数量统计 + 条形图展示
- 🌳 **木**：数量统计 + 条形图展示
- 💧 **水**：数量统计 + 条形图展示
- 🔥 **火**：数量统计 + 条形图展示
- 🟫 **土**：数量统计 + 条形图展示

**智能分析：**
- ✅ 五行齐全 → 运势稳定提示
- ⚠️ 五行缺失 → 补救建议（偏旁部首、幸运颜色）

**五行生克关系：**
```
相生：金→水→木→火→土→金
相克：金→木→土→水→火→金
```

#### 2.3 八卦命理分析

**本命卦计算：**
- 根据出生年月日计算先天八卦
- 显示卦名、卦符、卦象含义

**分析维度：**

| 卦名 | 乾/兑/离/震/巽/坎/艮/坤 |
| 卦符 | ☰/☱/☲/☳/☴/☵/☶/☷ |
| 卦象 | 自然象征（天/泽/火/雷/风/水/山/地） |
| 特质 | 性格特征描述 |
| 五行 | 卦象对应的五行属性 |

#### 2.4 纳音五行分析

**60 甲子纳音表查询：**

### 三、智能起名推荐

#### 3.1 推荐算法

**筛选流程：**
```
1. 字数过滤 → 2. 性别匹配 → 3. 五行补全 → 4. 古籍筛选 → 5. 评分排序
```

**评分规则：**
- 包含缺失五行元素：+30 分
- 性别关键词匹配：+20 分
- 古籍出处经典程度：+10-20 分
- 寓意美好程度：+10-30 分

#### 3.2 名字详细信息

每个推荐名字包含以下信息：

**基础信息：**
- 📝 **名字**：书法字体展示
- 📯 **拼音**：带声调符号
- 📚 **出处**：古籍原文引用
- 💡 **寓意**：含义解释
- ☯️ **五行**：名字字的五行属性
- 👦👧 **适用性别**：男/女/通用

**详细解析（可折叠）：**
| 分类 | 内容 |
|------|------|
| 📖 **易经解释** | 对应卦象及易经理念 |
| ☯️ **八卦解析** | 八卦属性及象征意义 |
| 🔮 **紫微斗数** | 对应星宿及影响 |
| 🎵 **纳音分析** | 纳音五行及含义 |
| 🔮 **命运走势** | 一生命运概述 |
| 💼 **事业前程** | 职业发展建议 |
| 💕 **婚姻感情** | 姻缘运势分析 |
| 🎓 **学业运势** | 学习考试运程 |

1. **下载项目**


## 🤝 使用说明

### 命名建议

1. **五行互补**：根据八字五行缺失，选择相应属性的字
2. **性别适配**：男孩选用阳刚之字，女孩选用阴柔之字
3. **音律和谐**：注意平仄搭配，读起来朗朗上口
4. **寓意美好**：选择积极向上的含义
5. **避免重名**：尽量避开过于常见的名字

### 文化常识

**天干五行：**
- 甲乙属木，丙丁属火，戊己属土，庚辛属金，壬癸属水

**地支五行：**
- 寅卯属木，巳午属火，申酉属金，亥子属水，辰戌丑未属土

**八卦象征：**
- 乾为天，坤为地，震为雷，巽为风，坎为水，离为火，艮为山，兑为泽
